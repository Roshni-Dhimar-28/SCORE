# 🚀 Score - Skill Based Student Freelancing Platform

A full-stack web application designed exclusively for students to showcase their skills, connect with recruiters, and work on freelance projects in a secure and collaborative environment.

---

## 🌟 Features

- 🔐 Role-based authentication (Students, Recruiters, Alumni)
- 📌 Project posting and bidding system
- 👩‍💻 Skill showcasing and profile management
- 🔥 Firebase authentication & real-time database integration
- 📊 Project tracking and updates

---

## 🛠️ Tech Stack

**Frontend:**
- React.js
- Tailwind CSS

**Backend:**
- Node.js
- Express.js

**Database & Auth:**
- Firebase (Firestore / Realtime DB)
- Firebase Authentication
